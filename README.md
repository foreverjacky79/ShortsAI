🎬 **YouTube Shorts AI 分析工具**
這是一款幫助您快速掌握 YouTube 趨勢並利用 AI 生成腳本提示詞 (Prompt) 的工具。

🔧 **第一步：環境準備 (最重要的步驟)**
由於本工具不包含龐大的影片處理組件，請您手動完成以下配置：
1. **下載 FFmpeg：** refer: https://ffmpeg.org/download.html
	
	  Windows: 請確保資料夾內除了 ShortsAIAnalyzer.exe 之外，還放了一個 ffmpeg.exe 檔案。

	  Mac: 請在終端機輸入 brew install ffmpeg 進行安裝。
2. **API 金鑰：**
  請準備好您的 YouTube Data API Key 與 Gemini API Key。

🚀 **第二步：開始搜尋趨勢**
1. **設定 API**：開啟程式後，先到「進階與 API」分頁填入金鑰並點擊搜尋。
2. **輸入關鍵字**：在「基本搜尋」中輸入關鍵字（例如：ASMR, Cooking）。
3. **觀察爆發指數**：程式會幫您計算哪部影片正在「爆紅」，指數越高代表成長越快。

💡 **第三步：新增功能 - 手動網址分析** (New!)
如果您在滑 YouTube 時看到一部很棒的 Shorts，不必在清單裡找，可以直接分析：
1. 切換到 「**AI Prompt 分析**」 分頁。
2. 在最上方的輸入框中，**直接貼上 Shorts 的網址**。
3. 點擊「**立即分析**」。

🧠 **第四步：解讀 AI 提示詞**
程式會下載影片片段並讓 Gemini AI 進行深度掃描：

 **鏡頭運動**：告訴您這部片是用什麼角度拍的。
 
 **光影氛圍**：分析這部片為什麼吸引視覺。
 
 **Prompt 生成**：直接給您一段英文提示詞，您可以將其複製到** Sora, Runway 或 Leonardo.ai** 產生類似風格的影片。

🔄 **第五步：自動更新 (如果您是使用 .py 檔案)**

如果您使用的是 .py 腳本，每次啟動時程式會自動連接到 GitHub：

若發現新版本，會詢問您是否更新。

點擊「是」之後，程式會自動完成升級並重啟，您永遠能用到最新的功能！

🛠️ **故障排除**

**分析失敗？** 請檢查您的 Gemini API Key 是否正確，或檢查網路是否穩定。

**無法下載？** 請確認 ffmpeg.exe 是否放在程式旁邊，且該資料夾沒有被防毒軟體鎖定。
