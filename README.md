# 🎬 YouTube Shorts 趨勢分析與 AI 助手

這是一款專為影音創作者設計的強大工具，結合了 **YouTube 數據抓取** 與 **Google Gemini AI**，協助您一鍵找出熱門短影音，並深度分析其視覺與腳本邏輯。

---

## 🛠️ 第一步：環境配置 (必看)

由於本工具採輕量化設計，請依您的作業系統進行簡易設定：

### 1. 取得 API 金鑰
* **YouTube Data API**: 用於搜尋影片數據。
* **Gemini API**: 用於 AI 影片分析。

### 2. 下載影片處理組件 (FFmpeg) https://ffmpeg.org/download.html
* **Windows 使用者**：請下載 `ffmpeg.exe` 並將其與 `ShortsAIAnalyzer.exe` 放在 **同一個資料夾**。
* **Mac 使用者**：請開啟終端機執行 `brew install ffmpeg`。



---

## 🚀 核心功能介紹

### 🔍 1. 爆發趨勢搜尋
* 在「基本搜尋」分頁輸入關鍵字。
* 設定搜尋天數（建議 3-7 天）以獲取最新趨勢。
* **爆發指數**：程式會自動計算觀看次數與時間比例，數值越高代表影片越火熱！

### ✨ 2. AI 影片提示詞 (Prompt) 分析
當您在清單中看到感興趣的影片時：
1. **點擊右鍵** (Mac 請用兩指點擊或 Ctrl+左鍵)。
2. 選擇 **「✨ 使用 AI 產生影片 Prompt」**。
3. 程式會自動下載片段並交由 Gemini 分析鏡頭、光影、動作與氛圍。



### 🔗 3. 手動網址分析 (新功能)
如果您在瀏覽器看到有趣的影片，可以直接分析：
* 切換至 **「AI Prompt 分析」** 分頁。
* 在上方網址列貼入 YouTube Shorts 連結並點擊 **「立即分析」**。

## ❓ 常見問題 Q&A

**Q: 點擊分析後程式沒反應？**
A: 請檢查「進階設定」中的 API Key 是否正確填寫。

**Q: 影片下載失敗？**
A: 請確認 `ffmpeg` 是否正確安裝或放置在程式同目錄下。

**Q: 可以在 Mac 上執行嗎？**
A: 可以！請確保安裝了 Python 並執行 `啟動工具.command`。
